The NvShade Philosophy

The Swift Shadow

Performance is paramount. A sub-50ms startup is the standard, achieved through disciplined, strategic lazy-loading via `lazy.nvim`.

The Forged Core

Built from strong, independent parts. The structure under `lua/forge/` is designed for maintainability and easy understanding.

Light in the Shade

Complexity is hidden until needed. The editor teaches you its features through discoverable keymaps (`which-key`) and a clear, consistent structure.

The Master Template

Provides a powerful default setup but makes it trivial to override any setting in a central `config.lua` file, allowing you to forge it into your own tool.

🚀 Installation

Prerequisites

  • Neovim v0.8.0+ (Install Guide)
  • A Nerd Font (e.g., FiraCode Nerd Font)
  • Git for installation and updates.
  • A C Compiler for `nvim-treesitter`.

Install NvShade

Choose your operating system and run the commands in your terminal.

🐧 Linux / macOS

1. Backup your existing configuration (important!):

mv ~/.config/nvim ~/.config/nvim.backup

2. Clone NvShade:

git clone https://github.com/shadowdevforge/NvShade.git ~/.config/nvim
🪟 Windows (Powershell)

1. Backup your existing configuration (important!):

Move-Item $env:LOCALAPPDATA\nvim $env:LOCALAPPDATA\nvim.backup

2. Clone NvShade:

git clone https://github.com/shadowdevforge/NvShade.git $env:LOCALAPPDATA\nvim

Finally, launch Neovim. On the first run, `lazy.nvim` will automatically install all plugins.

🎨 Customization

All your personal overrides belong in lua/forge/config.lua. This is the only file you should ever need to edit. This design ensures you can update NvShade with a git pull without losing your personal configurations.

-- Example: ~/.config/nvim/lua/forge/config.lua

local M = {}

M.plugins = {
  -- Override telescope layout
  {
    "nvim-telescope/telescope.nvim",
    opts = {
      defaults = {
        layout_strategy = "vertical",
      },
    },
  },
  
  -- Disable a default plugin
  { "goolord/alpha-nvim", enabled = false },
  
  -- Add a new plugin
  {
    "github/copilot.vim",
    event = "InsertEnter",
  },
}

return M

🩺 Troubleshooting

  • Missing Icons? Ensure you have installed and enabled a Nerd Font in your terminal's settings.
  • LSP Not Working? Run :LspInfo to check attachment, :Mason to verify installation, and :checkhealth for errors.
  • Slow Startup? Run :Lazy profile to identify potential bottlenecks.
  • Theme Not Loading? Run :Themery to select a theme, or check :Lazy to ensure plugins are installed.
