The NvShade Philosophy

Every feature, keymap, and design choice serves these guiding principles

The Swift Shadow

Performance First

NvShade must be imperceptibly fast. A sub-50ms startup is the standard, achieved through disciplined, strategic lazy-loading via lazy.nvim.

The Forged Core

Modularity

The configuration is built from strong, independent, and easily understood parts. The entire structure under lua/forge/ is designed for maintainability.

Light in the Shade

Intuitive Discovery

Complexity is hidden until needed. The editor teaches you its features through discoverable keymaps (which-key) and a clear, consistent structure.

The Master Template

Malleable

NvShade provides a powerful default setup but makes it trivial to override any setting in a central config.lua file, allowing you to forge it into your own perfect tool.

üöÄ Installation

Get started with NvShade in seconds. Requires Neovim v0.8.0+, a Nerd Font, Git, and a C compiler.

# 1. Backup your existing configuration
mv ~/.config/nvim ~/.config/nvim.backup

# 2. Clone NvShade
git clone https://github.com/shadowdevforge/NvShade.git ~/.config/nvim

# 3. Launch Neovim (plugins install automatically)
nvim
